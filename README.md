# NE555 Blinking Heart Circuit

This repository contains the design files for a blinking heart circuit using the NE555 timer IC.

## About the Project

The NE555 timer IC is utilized as an astable multivibrator in this project. The circuit creates a heart shape with LEDs that blink at a certain frequency. This simple and fun project is an important step in the electronic learning process.

## Files

- `Gerber`: The Gerber files required for PCB production.
- `Schematic`: The electronic schematic of the circuit.
- `3D_View`: The 3D view of the circuit.
